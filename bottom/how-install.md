# 🚀 Installing Bottom

**Bottom** is a system monitor (like htop for Windows).

## 📦 Installation
```powershell
winget install bottom
```
# ⚙️ Configuration
Copy the config:

Move the bottom.toml file to the folder: 
C:\Users\<Your user>\AppData\Roaming\bottom
Launch it:

```powershell
btm
```

# 🎯 Hotkeys

d — change mode.

q — quit.

↑↓←→ — navigate through processes.

Done!
